How to Extend

Adding New Components

To add new components to the playground:

  1. Generate a new component: ng generate component components/my-component
  2. Add the component to app.module.ts declarations
  3. Create an example section in this playground template
  4. Import any required Material or Kite modules
  5. View changes with ng serve
